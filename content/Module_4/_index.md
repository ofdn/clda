This chapter is on everyday reality (phones, memory cards, slow or breaking internet) with simple archiving habits to help long-term preservation.

# Trainer goal

Help participants keep their recordings safe, each file named clearly, and easy to find later, even with limited devices and connectivity.

# 4.1 Understand what you already have

Simple self-audit: list audio, video, photos already recorded in the community.
Group discussion: where are these stored now (phone, laptop, WhatsApp, social media)?
Risks: lost phones, deleted chat history, broken hard drives.

How to use, name and save files and folders.
How to make backups and keep a simple log/history of all files.
How to edit audio and video.
How to export in stable formats.
How to create transcripts or subtitles when possible. (see module_9])
